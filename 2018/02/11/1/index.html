Github Setup

Register a Github account anyway here

Create a new repository

  • Decide your Owner / Repository name
    as yourGithubName / yourGithubName.github.io
    Ex: mebestcoder / mebestcoder.github.io
  • just Public
  • just add a README friendly

Environment Setup

Open Terminal in your Mac and keep it open

Homebrew

Homebrew help you install some things that you need but Apple doesn’t.Paste those words on Terminal and you’ll get it.

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Git

  • Download for Mac OS X
  • Bind your Git to your GitHub account ( change the content inside “” ) 
    git config --global user.name "your Github Name"
git config --global user.email "your Github Register Email"
  • Generate a public SSH Key
    Check whether you have existed SSH Keys.It will list the files in your .ssh directory, if they exist.
    ls -al ~/.ssh
    If you don’t have an existing public and private key pair, then generate a new SSH key.
    ( change the content inside “” )
    ssh-keygen -t rsa -C “your Github Register Email”
    The public key is saved to id_rsa.pub.You can save it to the clipboard by directly running this:
    pbcopy < ~/.ssh/id_rsa.pub
    Back to your Github homepage and open github/settings/keys, add a new SSH key.
    Test whether it’s done in Ternimal:
    ssh git@github.com
    if it says “Hi xxx ( your Github Name )!You’ve successfully authenticated…”,congratulations!

Node.js ; npm

  • Hexo is based on Node.js.
    Download for Mac OS X ( current version )
  • Test the version of Node.js as well as npm ( supposed to be built-in ).
    Success when it tells you the version numbe.
    node -v
npm -v
  • Update npm.
    To avoid future referred errors that may appear,make sure to update npm to the latest version by this:
    npm install -g npm@latest

Hexo

  • Install Hexo with npm
    npm install hexo-cli -g
  • If you got some like
    “npm WARN checkPermissions ERR! Missing write access to /usr/local/lib/node_modules”
    you could try this:
    sudo chown -R `whoami` /usr/local/lib/node_modules
  • Let’s default you’ve done with the installation of Hexo.The next step is Creating a dir for Hexo to generate and store all static files to your blog automatically.
    How about just name your dir “blog”?
    hexo init blog
    cd blog
    npm install
    Bingo! Search your Mac and you’ll find a folder named “blog” under Users.

Configuration

Hexo Localhost Preview

  • First, cd your dir “blog” (change directory: moving from one directory to another)
    cd blog
  • Then use the following command to generate a static web page
    hexo generate 
#or
hexo g
  • Then you could ask the local server to display your page:
    hexo server
    #or
    hexo s
    The default local address is
    localhost:4000
    drop it into your browser, see what will happen.
